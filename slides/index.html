¿Qué es?

Es un cambio de los estilos de los elementos del DOM (Document Object Model).

¿Para qué sirven?

Dar dinamismo a los website o webapps para una mejor UI/UX.

Timing Functions

Es una funcion (matemática) que especifica la velocidad de una animación o transición

Cubic Bezier

Es una tecnica de dibujo técnico donde se utilizan formulas matemáticas, dentro de CSS tenemos la función 

                
                  cubic-bezier(x1, y1, x2, y2)

que recibe los punto de la curva con valores entre 0 y 1

transition property

Es una shorthand para poder usar transitiones en CSS

  • transition-property: propiedad que aplica la transición
  • transition-delay: retraso de la transición
  • transition-duration: duración de la transición
  • transition-timing-function: timing function de la transición

animation property

Es una shorthand para poder usar animaciones en CSS

  • animation-name: nombre de la animación
  • animation-duration: duración de la animación
  • animation-timing-function: timing function de la animación
  • animation-delay: retraso de la animación
  • animation-iteration-count: veces que se repite la animación
  • animation-direction: la dirección en la que sucede la animación
  • animation-fill-mode: modo en el que se queda el elemento luego de la animación
  • animation-play-state: estado de la animación

@keyframes

Son los fotogramas de referencia de una animación


              @keyframes identifier {
                [ from | to | percentage ] [, from | to | percentage ]* block ]*
              }

Ejemplo


              @keyframes bounce {
                0%, 20%, 50%, 80%, 100% {
                  transform: translateY(0);
                } 
                40% {
                  transform: translateY(-30px);
                } 
                60% {
                  transform: translateY(-15px);
                } 
              }
